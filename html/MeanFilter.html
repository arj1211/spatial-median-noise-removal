Apply Mean Filter on target image
Author: Arjun Bawa, Date last modified: Dec. 22, 2021
This function applies Mean filtering on the given image using the specified mask_size. It is defined as
The function is supposed to operate on an image matrix created by imread. When the function operates on an intake that is not of this expected form, it will throw an error. mask_size should be a odd integer, such that each pixel will be perfectly at the center of the mask. When the function operates on even integers, it will warn. However, mask_size should never be 0 or negative, in which case the function will error.
The function deals with edges in the following way: the mask will still center the pixel; if the pixel is too close to the edge such that parts of the mask went outside of the edge, those "outside of the edge" part is disregarded; the set of pixels used to determine the mean will only contain the valid pixels.
The function realizes MeanFilter, which is:
  • For each pixel in the image, place a mask centered at that pixel, the pixel and surrounding pixels included in the mask are extracted .
  • From the extracted pixels of concern, find the mean value of each RGB component, yielding a single 3-tuple .
  • The filtered resultant pixel is .
function filtered_img = MeanFilter(image, mask_size)
if (mask_size <= 0)
error('MeanFilter: must have a positive mask size');
end
img_size = size(image);
if (length(img_size) ~= 3 || img_size(3) ~= 3)
error('MeanFilter: the supplied image is not of a expected image form');
end
if (rem(mask_size, 2) == 0)
warning('MeanFilter: mask size is not odd, target pixel cannot be put at the center of the mask');
end
filtered_img = uint8(zeros(img_size));
for i = 1:img_size(1)
for j = 1:img_size(2)
mask = make_mask(image, [i j], mask_size);
filtered_img(i, j, :) = mean(mask);
end
end
end
