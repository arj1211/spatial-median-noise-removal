Apply Spatial Median Filter (SMF) on target image
Author: Arjun Bawa, Date last modified: Dec. 20, 2021
function filtered_img = SMF(img, mask_size)
img_size = size(img);
filtered_img = uint8(zeros(img_size));
for i = 1:img_size(1)
for j = 1:img_size(2)
mask = make_mask(img, [i, j], mask_size);
[~, idx] = min(sdepth(mask)); % note this line
filtered_img(i,j,:) = mask(idx, :);
end
end
end
