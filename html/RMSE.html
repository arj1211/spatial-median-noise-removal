Calculate Root Mean Square Error (RMSE) between two images
Author: Shizhen Li Date last modified: Dec. 4, 2021
This function is normally used to compare reconstructed image against original image. The sequence of the two image does not matter too much as norm will be used in calculation.
The function will throw an error if the two matrices don't have the same size. As the function is supposedly used on normal rectangular images (three dimensional matrix with the third dimension having a length of 3), the function will error if it is not the case.
The function realizes the following math formula (with additional error check):
function rms_error = RMSE(original_image, final_image)
dimensions = size(original_image);
% before hand error checks
if (dimensions ~= size(final_image))
error('RMSE: two params are not in the same size');
end
if (length(dimensions) ~= 3 || dimensions(3) ~= 3)
error('RMSE: the passed in param is not a supposed image matrix format');
end
% core algorithm part
% converts data to double for proper division results
ori_double = double(original_image);
fin_double = double(final_image);
% the 3D image matrix is flattened to make subtraction process easier.
% the special * 3 is to offset the fact that each pixel include 3 value.
rms_error = sqrt(mean((ori_double(:) - fin_double(:)).^2) * 3);
end
